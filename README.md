### Installation

#### Requirements

- Docker
- `docker-compose`
- Open ports:
  - `8000` - Client/API of dispatcher itself

#### Steps

1. Clone the repo
2. Build the images. In the root of the folder, run `docker-compose build`
3. Run the images. In the root of the folder, run `docker-compose up`
